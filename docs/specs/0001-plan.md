# The Oak and Barrel - Restaurant Website Implementation Plan

## Overview
Build a 3-page restaurant website for "The Oak and Barrel" - a family-friendly New York restaurant specializing in steaks, sushi, burgers, and craft beers with live music events.

**Tech Stack:** Next.js 16.1.6, React 19, Tailwind CSS v4, TypeScript

---

## File Structure

```
app/
├── layout.tsx              # Root layout (fonts, header, footer)
├── globals.css             # Tailwind theme from design system
├── page.tsx                # Home page
├── about/page.tsx          # About Us page
└── menu/page.tsx           # Menu page (book-like paging)

components/
├── layout/
│   ├── Header.tsx          # Navigation with reservation CTA
│   └── Footer.tsx          # Site footer
├── ui/
│   ├── Button.tsx          # Primary/Secondary buttons
│   ├── Badge.tsx           # Menu item badges
│   ├── Input.tsx           # Form inputs
│   └── Select.tsx          # Form selects
├── cards/
│   ├── MenuItemCard.tsx    # Menu item display
│   └── EventCard.tsx       # Event display
└── forms/
    └── ReservationModal.tsx # Reservation form modal

lib/
├── types.ts                # TypeScript interfaces
└── menuData.ts             # Parsed menu data with Pexels images
```

---

## Implementation Steps

### Phase 1: Configuration
1. **Update `next.config.ts`** - Add Pexels image domain
2. **Update `app/globals.css`** - Integrate design system from `docs/design/tailwind-theme.css`
3. **Update `app/layout.tsx`** - Add Google Fonts (Playfair Display, DM Sans, Cormorant Garamond)

### Phase 2: Data Layer
4. **Create `lib/types.ts`** - MenuItem, Event, ReservationData interfaces
5. **Create `lib/menuData.ts`** - Parse CSV data, map Pexels images, export categorized data

### Phase 3: UI Components
6. **Create `components/ui/Button.tsx`** - Primary/secondary/ghost variants
7. **Create `components/ui/Badge.tsx`** - Popular, Vegetarian, Spicy, etc.
8. **Create `components/ui/Input.tsx`** - Form input with error states
9. **Create `components/ui/Select.tsx`** - Form select dropdown

### Phase 4: Layout Components
10. **Create `components/layout/Header.tsx`** - Fixed nav, logo, reservation CTA, mobile menu
11. **Create `components/layout/Footer.tsx`** - Links, contact info, hours
12. **Create `components/forms/ReservationModal.tsx`** - Name, email, party size, date, time with validation

### Phase 5: Card Components
13. **Create `components/cards/MenuItemCard.tsx`** - Circular image, name, description, price, badge
14. **Create `components/cards/EventCard.tsx`** - Event image, title, date/time, description

### Phase 6: Home Page (`app/page.tsx`)
15. **Hero Section** - Full-screen background image, headline, CTA buttons
16. **Popular Items Section** - Grid of 4-6 items with "Popular" badge
17. **Upcoming Events Section** - Friday Jazz Night & Sunday Acoustic cards
18. **Reservation CTA Section** - Final call-to-action with brand background

### Phase 7: About Page (`app/about/page.tsx`)
19. **Page Hero** - Title and tagline
20. **Founder Story** - Kaka's story (started 2026), image, narrative
21. **Restaurant Gallery** - 3 ambiance images from Pexels
22. **Values Section** - Quality First, Family Welcome, Live Experience

### Phase 8: Menu Page (`app/menu/page.tsx`)
23. **Book-Like Container** - Two-page spread with visible spine
24. **Left Page** - Category name, chapter number, description
25. **Right Page** - Scrollable list of menu items with circular images
26. **Navigation** - Arrow buttons, category dots, quick nav tabs
27. **Page Flip Animation** - CSS 3D transforms for realistic turning
28. **Mobile View** - Single-column card layout with category tabs

## Key Design Elements

**Colors (from design system):**
- Primary: `#D4952B` (Golden Amber)
- Background: `#FDFCFA` (Warm Cream)
- Text: `#2A2520` (headings), `#5C5449` (body)

**Typography:**
- Headings: Playfair Display (serif)
- Body: DM Sans (sans-serif)
- Accents: Cormorant Garamond (italic)

**Menu Book UX:**
- Desktop: Two-page spread simulating physical menu
- Left page: Category intro with chapter number
- Right page: Items with circular images, descriptions, prices
- Navigation: Arrows + dots + category tabs
- Animation: Page flip with 3D transforms (600ms)
